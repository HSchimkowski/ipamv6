Information

System requirements

This Tool is written in languages, which are very popular and so you do not need vey special systems for it :)

Server

on the server side you need a Webserver running PHP5 an an MySQL Database.

If you want to protect the System agains unauthorized Access you can simply protect the published folder on your webserver with standard tool like .htaccess or something simmilar

All this Requirements comes with the most popular Linux distrubutions. Ex. Debian Linux.

Client

On Client-Side you only need a Webbrowser with enabled Javascript and HTML5 Support to access the Webserver.

This tool has been testet with Google Chrome and Mozilla Firefox, but other browsers schuld work as well.

installation

  1. Download application from http://ipamv6.schimkowski.net/download/IPAMv6.zip
  2. unzip and copy to your webserver
  3. import ipamv6 database-structure into MySQL
  4. change values into config.inc.php
  5. access your IPAMv6 instalation vith your webbrowser via http://[your.servers.address]/IPAMv6
