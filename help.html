Information

System requirements

This Tool is written in common web technologies, making it compatible with most standard systems.

Server

On the server side you need:

  • A web server running PHP5
  • MySQL Database

To protect the system against unauthorized access, you can secure the published folder on your webserver using standard tools like .htaccess.

All these requirements come with most popular Linux distributions (e.g., Debian Linux).

Client

On the client side, you only need:

  • A web browser with JavaScript enabled
  • HTML5 support

This tool has been tested with Google Chrome and Mozilla Firefox, but other modern browsers should work as well.

Installation

  1. Download application from http://ipamv6.schimkowski.net/download/IPAMv6.zip
  2. Unzip and copy to your webserver
  3. Import IPAMv6 database structure into MySQL
  4. Configure settings in config.inc.php
  5. Access your IPAMv6 installation via http://[your.servers.address]/IPAMv6
